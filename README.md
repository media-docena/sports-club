# Sports Club

A console application for a sports club that registers members and enrolls them in activities.


## How to run locally

Clone the project

```bash
  git clone https://sports-club
```

Go to the project directory

```bash
  cd sports-club
```

Execute in console

```bash
  dotnet run
```
